# Pageworker / 页工 IP

## 适用领域

- 知识整理、文档、笔记、Notion、知识库。
- 信息结构、内容系统、素材归档。
- 方法论梳理、文章结构、卡片化表达。
- 从混乱内容到清晰结构。

## 不适用领域

- 问题诊断、风险排查、故障现场。
- 强路径、强转化、强链路主题。
- 情绪化叙事或高冲突故事。

## 角色定义

页工是纸面工作流里的系统操作员。它负责把抽象信息折成结构，把混乱素材整理成可读卡片。

## 外形

- 身体像折页、索引卡、便签或薄纸盒。
- 主体是浅色纸页形状，用细黑线勾边，不能是黑色实心角色。
- 有一个小蓝色索引签，通常在头部或侧边。
- 可以有一枚小橙色纸夹、铅笔或封口贴作为工作道具。
- 眼睛是极简黑点或短线，表情冷静、认真、空白。
- 胳膊和腿像细纸条或细线，可以略微弯折。

## 性格

- 很认真，但做的事有点荒诞。
- 像一个低调的文档系统操作员。
- 冷幽默，不卖萌。
- 笨拙但可靠。

## 动作库

- 折叠复杂流程。
- 给信息卡片贴签和归档。
- 把素材塞进纸夹机器。
- 用蓝色索引签标出关键判断。
- 把混乱内容压平成一张卡。
- 称量一张卡片是否足够重要。
- 封住一个漏口或打开一个新入口。

## 道具库

- 索引卡、便签、文件夹、抽屉、纸夹、铅笔、封口贴、打孔器、卡槽、纸盒。

## 禁止

- 不要画成黑色实心小怪物。
- 不要画成动物、机器人、儿童卡通或表情包。
- 不要给页工复杂服装、闪亮眼睛或夸张表情。
- 不要让页工只是站在角落里看。
- 不要把蓝色索引签扩大成主角；它只是识别点。
