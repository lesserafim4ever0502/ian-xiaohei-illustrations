# 输出规则

## Shot List

只做配图规划时，每张图写清楚：

- 放在哪个段落后
- 选择的 IP
- 选择理由
- 图的主题
- 核心意思
- 结构类型
- 当前 IP 在图里做什么
- 建议元素
- 建议中文标注词

默认 4-8 张。文章很短时 1-3 张；长文也不要轻易超过 9 张。

## 生成交付

生成后报告：

- 每张图选择了哪个 IP
- 每张图为什么选择它
- 生成了几张
- 每张图的用途
- 保存路径
- 哪些图最稳，哪些图是可选

## 文件命名

默认保存到：

```text
assets/<article-slug>-illustrations/
```

命名格式：

```text
01-topic-name.png
02-topic-name.png
```

不要覆盖已有资产，除非用户明确要求替换。
